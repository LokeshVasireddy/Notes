ReactDOM
-----------
ReactDOM is the package that connects React to the browser DOM.
What it does
Takes React components
Renders them into real HTML elements
Updates the DOM efficiently

React.StrictMode
----------------
React.StrictMode is a development-only tool that helps you find bugs early.
What it does
Runs extra checks
Warns about unsafe patterns
Intentionally re-renders components to expose bugs
