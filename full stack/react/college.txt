Usestate
-------
useState is a React Hook that lets a functional component store, read, and update state.
const [count, setCount] = useState(0);
setCount(prev => prev + 1);

const [user, setUser] = useState({
  name: "Loki",
  age: 25
});

setUser(prev => ({
  ...prev,
  age: prev.age + 1
}));

useRef
------
useRef lets you store a mutable value that persists across renders without causing a re-render.
import { useRef } from "react";

const ref = useRef(initialValue);
Returns an object: { current: initialValue }

The object stays the same between renders




useReducer
----------
useReducer is a Hook used to manage complex state logic using a reducer function.

const reducer = (state, action) => {
  switch (action.type) {
    case "INC": return { count: state.count + 1 };
    case "DEC": return { count: state.count - 1 };
    default: return state;
  }
};

const [state, dispatch] = useReducer(reducer, { count: 0 });
