1.priority queue
--------------

| Function  | Meaning                             |
| --------- | ----------------------------------- |
| `push(x)` | Add element                         |
| `pop()`   | Remove the highest priority element |
| `top()`   | View highest priority element       |
| `empty()` | Check if empty                      |

int main() {
    priority_queue<int> pq;

    pq.push(10);
    pq.push(30);
    pq.push(20);

    while (!pq.empty()) {
        cout << pq.top() << " "; // prints: 30 20 10
        pq.pop();
    }
}

map<string, int> mp;

| Operation            | Code                       | Complexity |
| -------------------- | -------------------------- | ---------- |
| Insert               | `mp[key] = value;`         | O(log n)   |
| Insert (alternative) | `mp.insert({key, value});` | O(log n)   |
| Access value         | `mp[key]`                  | O(log n)   |
| Erase key            | `mp.erase(key);`           | O(log n)   |
| Check if exists      | `mp.count(key)`            | O(log n)   |
| Size of map          | `mp.size()`                | O(1)       |
| Clear                | `mp.clear()`               | O(n)       |
