Idempotency means:

Performing the same operation multiple times produces the same result as performing it once.

| HTTP Method | Idempotent? | Reason                                     |
| ----------- | ----------- | ------------------------------------------ |
| **GET**     | ‚úÖ Yes       | Only fetches data                          |
| **PUT**     | ‚úÖ Yes       | Replaces resource completely               |
| **DELETE**  | ‚úÖ Yes       | Deleting again has no effect               |
| **POST**    | ‚ùå No        | Creates new resource each time             |
| **PATCH**   | ‚ùå Usually   | Partial update may change state repeatedly |
